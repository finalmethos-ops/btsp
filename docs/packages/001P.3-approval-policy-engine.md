# Implementation Package 001P.3 — Approval Policy Engine

## Objective

Move BPP purchasing approval selection into configurable, testable policy evaluation.

## Scope

This package adds:

- Typed approval input, match, result, and level contracts
- Deterministic approval-level ranking
- Configuration-backed threshold, vendor, category, and default matching
- Highest-required-approval selection
- Safe disabled and incomplete-configuration behavior
- Immutable positive-evaluation snapshots
- Three BPP policy permissions
- Six approval policy configuration defaults
- Repeatable approval-default seed behavior
- Protected evaluation, defaults, and seed endpoints
- Database-backed policy and snapshot tests
- Approval policy workflow documentation

## Permissions

- `workflow.bpp.regional_approve`
- `workflow.bpp.policy_manage`
- `workflow.bpp.policy_read`

Core BPP transition permissions continue to be used by the policies that route to department, purchasing, and executive approval.

## API Surface

- `POST /api/v1/approval-policies/evaluate`
- `GET /api/v1/approval-policies/bpp-purchasing/defaults`
- `POST /api/v1/approval-policies/bpp-purchasing/seed-defaults`

## Architecture Rules Supported

- Approval thresholds and restricted lists are configuration-backed.
- BPP approval behavior remains isolated from Independent ordering.
- API-submitted actors are replaced with authenticated identity.
- Positive policy decisions create append-only snapshots.
- Missing enabled-policy configuration fails without silently approving a request.

## Validation Commands

```bash
cd backend
alembic upgrade head
pytest
ruff check app tests
```

## Definition of Done

- Executive and regional thresholds match inclusively.
- Executive outranks regional when both match.
- Restricted vendors and categories require purchasing review.
- Department review is the enabled default.
- Disabled policy returns no approval requirement.
- Positive evaluations write the required snapshot payload.
- Incomplete required configuration fails safely.
- Migration, tests, and Ruff pass cleanly.
